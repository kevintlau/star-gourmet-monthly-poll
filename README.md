# Star Gourmet - Monthly Poll

A companion web application that allows users to vote on recipes to be included in the monthly game update.

### Technologies Used

* HTML5
* CSS3
* JavaScript
* Node
* Express
* MongoDB (w/ Mongoose)
* OAuth (w/ Google login)
* Bootstrap
